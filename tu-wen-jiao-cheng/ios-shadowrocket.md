# iOS - Shadowrocket

如果有自己的美区账号，则可以直接搜索Shadowrocket，购买并下载。

{% hint style="info" %}
影猫为用户提供美区账号，并且已购买Shadowrocket，用户可以免费下载安装。
{% endhint %}

{% hint style="success" %}
请在用户中心开工单索取美区账号
{% endhint %}

![](../.gitbook/assets/ss-ios-1.png)

### 获取订阅链接

打开影猫官网，在[用户中心](https://sscat.me/user)可以查看自己的订阅链接，点击拷贝。​

![](https://blobscdn.gitbook.com/v0/b/gitbook-28427.appspot.com/o/assets%2F-LLrIfszdGKexROFMbHj%2F-LMVoxMIpHTaoD-TS7XM%2F-LMVw0Qs44UO9oyuajGt%2Fdingyue.png?alt=media&token=873593e3-6c60-4bd8-aeb7-de2dc7e7954d)

### 将订阅链接导入客户端 {#jiang-ding-yue-lian-jie-dao-ru-ke-hu-duan}

打开Shadowrocket，在首页点击右上角的＋号，添加配置

![](../.gitbook/assets/ss-ios-2.png)

添加节点类型为“Subscribe”，在URL栏目填入订阅链接，最后点击右上角完成。

![](../.gitbook/assets/ss-ios-3.png)

回到APP首页，服务器节点已导入，选择任一节点，点击开关即可连接！延迟测试按钮可以测试节点的延迟，作为参考。

![](../.gitbook/assets/ss-ios-4%20%281%29.png)

{% hint style="info" %}
至此基本配置已完成，系统栏出现VPN图标。请测试能否访问[**google.com**](https://www.google.com)\*\*\*\*
{% endhint %}

### 全局路由

* 配置：按照配置文件规则访问，默认配置是国内直连、国外代理；可以在配置中修改规则。
* 代理：全部流量走代理。当配置规则无法打开某一网站时，可以尝试全局代理。

![](../.gitbook/assets/iosnew.png)

### 自动更新订阅

在设置页面，点击“服务器订阅”，打开自动更新开关，即可在客户端启动时自动更新服务器订阅

![](../.gitbook/assets/iosnew2.png)

### 添加到widget（小组件）

推荐将shadowrocket添加至通知中心的widget，可以实现一键开关、查看实时网速等功能。

![](../.gitbook/assets/img_5602.jpg)
