# iOS - Quantumult

Quantumult是一款优秀的代理工具，界面更加炫酷！

![](../.gitbook/assets/quan.png)



如果有自己的美区账号，则可以直接搜索Quantumult，购买并下载。

{% hint style="info" %}
影猫为用户提供美区账号，并且已购买Quantumult，用户可以免费下载安装。
{% endhint %}

{% hint style="success" %}
请在用户中心开工单索取美区账号
{% endhint %}

### 获取订阅链接

打开影猫官网，在[用户中心](https://sscat.me/user)可以查看自己的订阅链接，点击拷贝。​

![](https://blobscdn.gitbook.com/v0/b/gitbook-28427.appspot.com/o/assets%2F-LLrIfszdGKexROFMbHj%2F-LMVoxMIpHTaoD-TS7XM%2F-LMVw0Qs44UO9oyuajGt%2Fdingyue.png?alt=media&token=873593e3-6c60-4bd8-aeb7-de2dc7e7954d)

### 将订阅链接导入客户端 {#jiang-ding-yue-lian-jie-dao-ru-ke-hu-duan}

打开Quantumult，在设置选项卡，点击`订阅`，再点击右上角＋号，添加订阅。

![](../.gitbook/assets/qu1.png)

将影猫订阅链接粘贴至输入框，并随意填写`名称`（方便自己区分），最后点击保存。

![](../.gitbook/assets/qu2.png)

保存后，客户端会导入服务器节点信息，可以在设置-服务器中查看。

![](../.gitbook/assets/img_7825.PNG)

### 开始连接

![](../.gitbook/assets/qu3.png)

如上图所示，开始连接

{% hint style="info" %}
连接方式的说明

* 全局代理： 全部流量走代理
* 自动分流：根据设定的规则自动分流，默认配置是国内直连、国外代理
* 全局直连：全部流量不走代理
{% endhint %}
